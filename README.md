# Agora-bot
## About
Firefox extension showing good answers of agora-quiz's quiz
## Installation
- Clone this repo: ```git clone https://github.com/CyrilleStr/agorabot```
- Go to ```about:debugging#/runtime/this-firefox``` and click on ```Load Temporary Add-on...```
- Select ```agorabot.js```
- Reload Agoraquiz
- It's ready !
